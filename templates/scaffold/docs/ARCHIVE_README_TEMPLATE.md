# Archived Documentation

This folder contains obsolete, superseded, or completed documentation.

## Archive Convention

**Naming:** `YYYY-MM-DD-<topic>/` for folders, `YYYY-MM-DD-<topic>.md` for files

**What goes here:**
- Completed plans (after all checkboxes done)
- Superseded specs/designs
- Historical reference docs
- Deprecated features documentation
