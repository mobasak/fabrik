## Plugin suggestions from THIS list for your website presets (saas / company / content / landing)

### Common baseline (use across all 4)

* **WP Mail SMTP** (or **Postman SMTP Mailer/Email Log**) — reliable email sending.
* **UpdraftPlus** — backups.
* **Wordfence Security** (or **iThemes Security**) — security layer.
* **WP Rocket** (or **W3 Total Cache**) + **WWW Image Optimizer** (or **Kraken.io**) — speed.
* **Google Tag Manager** — tracking codes in one place.
* **Redirection** — clean 301 management.

### `landing`

* **SeedProd Coming Soon Pro** (if pre-launch) or **OptimizePress** (if you want funnels inside WP).
* **Hello Bar** (simple CTA bar/popup).
* **WP Mail SMTP** + **Google Tag Manager** + **WP Rocket** (priority).

### `company`

* **WP Testimonials** (or “Testimonials WordPress Plugin”) — trust.
* **InstaShow** (only if Instagram/social proof matters).
* **Breadcrumb NavXT** (if theme lacks breadcrumbs).

### `content`

* **WP Time To Read** + **TLDR** — engagement and skimmability.
* **Ad Inserter** (only if monetizing with ads).
* **WordPress Popular Posts** — internal recirculation.

### `saas`

* **Go Pricing** — pricing tables.
* **Bookly** (only if you sell bookings/consulting).
* **S2Member** (only if gated membership/subscriptions are core).
